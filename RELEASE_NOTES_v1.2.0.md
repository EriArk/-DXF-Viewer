# DXF Viewer v1.2.0

## Editions

### DXF Viewer (Standard)
- Open DXF from file dialog, drag-and-drop, or recent files
- Fast 2D viewing with zoom/pan
- Light/Dark themes
- Millimeter rulers and guide-based measurement (`ΔX`, `ΔY`)
- Line thickness control

### DXF Viewer Plus
Includes all Standard features, plus:
- Open folder workflow for DXF collections
- Recent folders support
- Sidebar panel with all DXF files in the active folder
- Quick file switching inside the active folder

## Changes Since v1.1.0
- Introduced two release editions: Standard and Plus
- Added edition-aware UI and behavior switching
- Kept folder-centric workflow in Plus edition only
- Removed duplicated app-name chip in the top bar
- Fixed build packaging filters to prevent oversized `.deb` artifacts

## Linux Packages
- `dxf-viewer_1.2.0_amd64.deb` (Standard)
- `dxf-viewer-plus_1.2.0_amd64.deb` (Plus)
